杂感
=================

**算法到底是有用还是没用?** 这个问题一直萦绕在我的心里. 根据事实是检验真理的唯一标准来说, 众多的公司都选择算法作为面试重点之一, 这足以说明算法的重要性. 但是直到自己做一些项目中,发现自己敢于去尝试复杂的逻辑了, 此时我才相信算法的魅力.
